# Victorian Plumbing Listing Page

A quick prototype of a product listing page built with React, TypeScript, Vite, and Tailwind CSS.

## Quick Start

### Prerequisites

- Node.js 18+ and pnpm

### Installation

1. Clone the repository and navigate to the project directory:

```bash
cd victorian-plumbing-lucas
```

2. Install dependencies:

```bash
pnpm install
```

3. Start the development server:

```bash
pnpm dev
```

The application will be available at `http://localhost:5173`

## Tech Stack

- **React 19**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **Radix UI**
- **TanStack React Query**
- **Axios**
- **ESLint**
